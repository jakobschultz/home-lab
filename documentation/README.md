# Documentation

This folder contains design documents, planning notes, standards, and reference material for the homelab.

Use this folder to explain what the environment is supposed to look like and why design choices were made.

This folder should include:
- Network design
- VLAN plan
- IP addressing plan
- Naming standards
- Security model
- Backup strategy
- Monitoring plan

Actual device configs belong in `configs/`.
Step-by-step procedures belong in `runbooks/`.
